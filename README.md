# System Biblioteczny
System biblioteczny - aplikacja webowa, służąca do zarządzania biblioteką oraz obsługi czytelników. System umożliwia przechowywanie informacji o książkach, autorach i użytkownikach, a także obsługę wypożyczeń, zwrotów i rezerwacji książek. Aplikacja automatyzuje procesy biblioteczne, ułatwia kontrolę dostępności egzemplarzy oraz wspiera zarządzanie karami za przetrzymanie książek. 

# Funkcjonalności:
- Rejestracja i logowanie użytkownika 

- Wyszukiwanie książek po tytule bądź autorze 

- Filtrowanie książek po gatunkach 

- Informacje o dostępności książek 

- rezerwacja książek, możliwość przedłużenia rezerwacji 

- historia wypożyczeń 

- kary pieniężne za niedotrzymywanie terminów zwrotów
